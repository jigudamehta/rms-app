Host on GitHub Pages Fixes Google Sheets sync
Running from file:// or OneDrive breaks Google Sheets sync because the browser strips POST request bodies for security. Hosting on GitHub Pages (free, 5 minutes) fixes this permanently.
1Go to github.com → Sign in (free account)
2Click New Repository → Name it rms-app → Public → Create
3Click uploading an existing file → drag your HTML file → Commit
4Go to repo Settings → Pages → Source: Deploy from branch → main → Save
5Your app is live at: username.github.io/rms-app/rms_super_app_v11.html
✓Also update GAS Script A to v4.0 (adds CORS headers) → redeploy
